# Options

参数|说明|类型|可选值|默认值
---|:--:|---:|---:|---:
icon|选择图标按钮的内容|String|settings,error,info,loading,download,thumbs-up,down,right,left|---
loading|显示是否加载状态|Boolean|false，true|false
icon-position|选择图标的位置|String|left,right|left
